#  Simple Bank Management System (C# Console App)

A simple *Bank Account Management System* built using *C#* and *File Handling*.
This project simulates basic banking operations such as account creation, deposit, withdrawal, transfer, and transaction history.

---

##  Features

*  Account Creation with password protection
*  Secure Login System
*  Deposit Money
*  Withdraw Money
*  Transfer Money between accounts
*  Transaction History
*  Change Password
*  Persistent data storage using text files

---

##  Technologies Used

* *Language:* C#
* *Type:* Console Application
* *Concepts:*

  * Object-Oriented Programming (OOP)
  * Encapsulation
  * File Handling
  * Lists & String Manipulation

---


####  How It Works

### 1. Account Creation

* User provides name, password (min 8 characters), and initial deposit (min 500 taka)
* System generates a unique account number

### 2. Login

* User logs in using account number and password

### 3. Banking Operations

After login, user can:

* Check balance
* Deposit money
* Withdraw money
* Transfer money
* View transaction history
* Change password

---

##  Data Storage

All data is stored locally using text files:

* AccountList.txt
* Personalinfo.txt
* Transaction.txt

## This ensures data persistence even after program restart.
